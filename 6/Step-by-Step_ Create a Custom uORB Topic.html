Create a Custom uORB Topic

From definition to implementation in 6 steps

1

Define Message File msg/MyTopic.msg

# My custom topic
uint64 timestamp # required
float32 value
uint8 ORB_QUEUE_LENGTH = 4
4

Include Generated Header

Build system auto-generates C/C++ struct from .msg

#include <uORB/topics/my_topic.h>
2

Register Message

Add filename to build system

msg/CMakeLists.txt:

set(msg_files
  ...
  MyTopic.msg
)
5

Publish Data

uORB::Publication<my_topic_s> pub{ORB_ID(my_topic)};
my_topic_s msg{};
msg.timestamp = hrt_absolute_time();
pub.publish(msg);
3

Build Firmware

Run make command to generate headers.

make px4_fmu-v6x_default
6

Subscribe Data

uORB::Subscription sub{ORB_ID(my_topic)};
my_topic_s data;
if (sub.update(&data)) { ... }
