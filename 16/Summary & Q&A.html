Key Takeaways

Modular & Reactive Architecture

PX4 uses uORB for asynchronous pub/sub messaging, enabling decoupled and parallel module execution on NuttX RTOS.

uORB is the Nervous System

Create custom messages in msg/ to extend functionality. Use thread-safe shared memory for high-rate data.

Runtime: Tasks vs Work Queues

Prefer Work Queue Items for most modules (efficient, cooperative). Use Tasks only for blocking or isolated workloads.

VTOL Complexity

Leverage transition logic (vtol_att_control) and energy management (TECS). Validate transitions in SITL first.
