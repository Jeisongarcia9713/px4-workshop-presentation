uORB: The Nervous System

Asynchronous Pub/Sub API

Micro Object Request Broker. Decouples modules so they communicate without direct dependencies. Publishers advertise data; subscribers listen for updates.

Thread-Safe Shared Memory

Zero-copy mechanism for efficient data transport between threads and processes. Crucial for high-frequency sensor data (IMU @ 1kHz).

Reactive System Design

Enables event-driven architecture. WorkQueue modules can "wake up" only when a subscribed topic (e.g., vehicle_attitude) updates.

Introspection Tools

Built-in CLI commands for debugging. Inspect system health, message rates, and content in real-time on the console.
