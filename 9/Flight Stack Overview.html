The Flight Stack

A strictly ordered pipeline transforms raw sensor data into precise actuator commands through estimation and cascaded control loops.

Cascaded Control

Position → Velocity → Attitude → Rate. Each outer loop provides setpoints to the inner loop, running at increasing frequencies.

Commander & Navigator

High-level modules handle state machine transitions (Armed/Disarmed, Modes) and mission logic, feeding setpoints to controllers.

Control Allocation

Decouples control logic from airframe geometry. Controllers request physical forces/torques; allocator maps them to available motors/servos.

TECS (Fixed-Wing)

Total Energy Control System manages potential (altitude) and kinetic (airspeed) energy balance using throttle and pitch.
