Key Development Areas

Key Development Areas

src/modules/

Flight control modules, estimators, controllers. Where most application logic lives.

src/drivers/

Hardware abstraction layer for sensors, actuators, and communication interfaces.

msg/

uORB message definitions for inter-module communication.

src/lib/

Shared libraries and utilities used across multiple modules.

src/systemcmds/

System commands and utilities for debugging and configuration.

Understanding the repository structure is essential for efficient PX4 development
Start with src/examples/ for learning
